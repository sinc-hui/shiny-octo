# Personal CRM - 人脉管理工具

一个简洁的个人通讯录和客户关系管理工具。

## 功能特点

- 📇 联系人管理：增删改查、标签分组、关系热度追踪
- 💬 沟通记录：电话、会面、微信、邮件、工作往来全记录
- ✅ 跟进任务：设置优先级和截止日期，逾期提醒
- 🔍 智能搜索：支持姓名、公司、标签搜索
- ✨ AI 增信建议：基于互动记录自动生成关系维护建议

## 使用方式

直接打开 `index.html` 即可使用，数据存储在浏览器本地。

## 在线访问

https://你的用户名.github.io/personal-crm/

## 技术栈

纯 HTML + CSS + JavaScript，无需后端，数据存储在 localStorage。
